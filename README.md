# Welcome to Basemaster 
### Importing the Library
**Step 1. Add the JitPack repository to your build file**  

    allprojects {  
          repositories {  
             ...  
             **maven { url 'https://jitpack.io' }**  
          }   }

 
     
**Step 2. Add the dependency**  

    dependencies {  
       **compile 'com.github.tplloi:basemaster:1.0.3'**  
    }
